【基础题】
1.将1024和2048的乘积输出到控制台中。
2.使用弹窗输入框获取一个整数值t，将整数t的5次方的值输出到网页中。
3.使用弹窗输入框获取一个整数值k，如果k是偶数就输出“偶数”到网页中，否则输出“奇数”到网页中。
4.使用弹窗输入框获取一个1到7之间的整数值n，如果n为1就输出“星期一”到网页中，如果n为2就输出“星期二”到网页中，...以此类推。
5.使用弹窗输入框获取两个整数a和b，其中a一定比b的值要小。将a和b之间能被3整除的整数输出到网页中。
6.使用弹窗输入框获取三个整数a、b、c，输出这三个整数中最大的值到网页中。
7.使用弹窗输入框获取四个整数a、b、c、d输出这四个整数中最小的值到网页中。
【进阶题】
8.自定义一个函数，使其能够对给定的整数数组按从小到大排序，并将排序后的数组作为结果进行返回。
9.自定义一个函数，使其能够在给定的参数中找到最大值，并将最大值作为结果进行返回。（提示，可以使用arguments）
10.自定义一个函数，使其能够将给定整数的二进制表达，转换为相应的十进制数，并返回该十进制数。
举例
输入：001
输出：4
输入：1001
输出：9
输入：01101
输出：22
