【基础题】
1.使用弹窗输入框获取一个字符串，如果这个字符串能够转换为一个浮点数则输出“成功转换为浮点数”到网页中，否则输出“非浮点数”到网页中。

2.使用弹窗输入框获取一个字符串，将该字符串输出到网页中，并且换行后将该字符串的长度也输出到网页中。

3.使用弹窗输入框分别获取两个整数a和b，将这两个整数中值更大的那个输出到网页中。

4.使用弹窗输入框获取一个字符串，将字符串中所有的数字按顺序输出到网页中。

5.使用弹窗输入框获取一个字符串，将该字符串中数字的个数输出到网页中。

6.使用弹窗输入框获取一个整数值t，输出1+2+3+...+t的值到网页中。

7.使用弹窗输入框获取一个字符串，将该字符串中的数字转换为数值，并输出所有数字转换为数值后的总和到网页中。

【进阶题】
8.使用弹窗输入框获取一个整数值k，随后输出k行星号。第一行星号的个数为k个，随后往后每行递减一个星号。

9.使用弹窗输入框获取一个整数值n，输出1到n之间（包括n）所有能被3整除的数到网页中，换行后将它们的和输出到网页中。

10.使用弹窗输入框获取一个字符串，输出字符串中“aaa”的个数到网页中。
	举例》
	输入：aaajfkslfaakjljfsdaaa123lfjdklaaaasdf
	输出：3
	输入：falaaajdflasfaaasdfjlsdkaaasdfjskldjfaaaaafda
	输出：4
