【基础题】
1.将“我与赌毒不共戴天”输出到网页中，并且将字体设置为加粗、居中、红色。
2.使用弹窗输入框获取一个整数n，输出2 * n的值到网页中。
3.使用弹窗输入框获取一个字符串，如果这个字符串能够转换为整数则输出“整数”到网页中，否则输出“非整数”到网页中。
4.使用弹窗输入框获取一个字符串，将这个字符串的长度和第1个字符输出到网页中。
5.使用弹窗输入框获取一个整数，在网页中输出这个整数是几位数，个位输出1、十位数输出2、百位数输出3、以此类推。
6.使用弹窗输入框获取一个整数k，如果整数k为质数则输出“质数”到网页中，否则输出“非质数”到网页中。
7.输出1到100内所有的质数到网页中。
【进阶题】
8.设计并实现一个自定义函数，可以将给定字符串中下标为偶数的字符取出，并按顺序将这些字符拼接成新的字符串后返回。
举例
输入：abcdefg
输出：bdf
9.设计并实现一个自定义函数，其找出指定范围内的所有偶数，并按顺序放入数组中，最后返回。
10.设计并实现一个自定义函数，其可以将给定的字符串，按字母ASCII码值的顺序，对字符串中的字符进行从小到大重新排序，并返回排序后的结果。
