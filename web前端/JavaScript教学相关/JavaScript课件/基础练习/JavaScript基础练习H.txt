【基础题】
1.输出一个边长为100px的绿色正方形到网页中。
2.输出一个边长为100px的红色圆形到网页中，并设置其居中。
3.使用弹窗输入框获取一个字符串，输出该字符串到网页中。
4.使用弹窗输入框获取一个字符串，输出该字符串的长度到网页中。
5.使用弹窗输入框获取一个字符串，输出该字符串的前5个字符到网页中。
6.使用弹窗输入框获取一个正整数k，输出正整数k的立方值到网页中。
7.使用弹窗输入框获取两个正整数a和b，输出这两个数中值最小的数到网页中。
【进阶题】
8.使用弹窗输入框获取三个不重复的字符，输出这三个字符的所有组合到网页中，每种不同的组合用、隔开。
9.使用弹窗输入框获取一串数字字符串，将该字符串中的数字按从小到大排序后输出到网页中。
10.使用弹窗输入框获取一串字符串，将该字符串中所有的数字字符变成*，如何输出到网页中。
