【基础题】
1.输出1到100内所有奇数的和到网页中。

2.输出1到100内所有奇数的个数到网页中。

3.输出1到100内所有奇数的平均值到网页中。

4.使用弹窗输入框获取一个整数k，输出1到k内所有奇数的平均值到网页中。

5.使用弹窗输入框获取一个字符串，输出该字符串中下标为偶数的所有字符到网页中。

6.使用弹窗输入框获取一个整数值k，输出100到1000内小于k的所有整数的和到网页中。

7.输出三个红色圆到网页中，并将这三个圆放置到同一行，且将它们都居中。

【进阶题】
8.使用弹窗输入框方便获取两个字符串a和b，将a字符串中下标为奇数的字符与b字符串中下标为奇数的字符串进行交换。将两个交换后的字符串，输出到网页中。
	举例
	输入：abcdefg
		  1234567
	输出：1b3d5f7
		  a2c4e6g
	
9.使用弹窗输入框输入一个字符串，将字符串中所有的小写字母变成大写字母，并将转换后的结果输出到网页中。

10.使用弹窗输入框获取一个整数值k，输出k转换为二进制后的值到网页中。
	举例 
	输入：1
	输出：1
	输入：2
	输出：01
	输入：13
	输出：1011
